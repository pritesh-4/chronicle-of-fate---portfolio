# ⚔️ The Last Chronicle

> _"This story is just for that one reader."_

An immersive portfolio experience inspired by **Omniscient Reader's Viewpoint (ORV)**, where achievements become timelines, projects become scenarios, and software engineering becomes a story.

Unlike traditional portfolios, **The Last Chronicle** is designed as an interactive narrative that guides visitors through my journey as a developer, open-source contributor, and builder.

## 📖 About The Project

The Last Chronicle transforms a standard portfolio into an ORV-inspired interactive experience.

Visitors begin by accepting a scenario, then journey through:

- Personal origins
- Technical attributes
- Achievement timelines
- Project archives
- Hidden scenarios
- Future ambitions
- The final chapter

Every section is designed to feel like a chapter from an unfinished chronicle.

## 🚀 Getting Started

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## 🎨 Design Philosophy

Most portfolios answer:

> What can this developer do?

The Last Chronicle attempts to answer:

> How did this developer become who he is?

The project combines storytelling, animation, and frontend engineering to create an experience that feels less like a portfolio and more like a journey through interconnected timelines.
